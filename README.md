# EduAid: Unified Scholarship Portal

EduAid is a web-based platform designed to **help Filipino tertiary students discover, track, and apply for scholarships efficiently**. The portal centralizes opportunities from multiple sources, streamlines the application process, and provides automated reminders for deadlines, ensuring students never miss an opportunity.

## Key Features
- **Scholarship Aggregation:** Collects and organizes scholarships from various providers.
- **Application Tracking:** Monitor deadlines, requirements, and submission status.
- **Student Dashboard:** Personalized view of applied scholarships and saved opportunities.
- **Secure Account Management:** Ensures privacy and security of user data.

EduAid aims to reduce the hassle of searching for scholarships manually, helping students focus on their studies while maximizing access to financial aid opportunities.
